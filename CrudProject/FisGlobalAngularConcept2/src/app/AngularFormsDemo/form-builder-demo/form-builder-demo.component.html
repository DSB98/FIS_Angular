FORM BUILDER

In Angular two ways to creating angular reactive forms. By creating explicitly forms by form group and form control class using new keyword.

Another way FormBuilder class. This class to provide sugar coating or shorten , simpler way to create an object of from group, form array

NOTE : Formbuilder class reduces amount of code, we uses to write , to build reactive forms

The formbuilder class has three methods
Control() :Control method usae to create / construct new form control
Group() : Group is use to create new form group
Array() : Create new array form
